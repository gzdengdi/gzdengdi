Java中的对象类型转换--向上/向下转型

发表于 2017-04-18 | 分类于 技术 ， Java



概念定义

向上/向下转型是一种对象类型转换，在继承的基础上出现。目的是为了让类的使用范围和适用范围发生变化，以便操作不同范围的变量或者方法。
